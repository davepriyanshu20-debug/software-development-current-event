# Member 2 Research Notes

## My Role
My role is to research and explain the benefits and impact of GitHub Copilot on software development.

## Benefits of GitHub Copilot
- GitHub Copilot suggests code automatically while developers type.
- It helps save time by reducing repetitive coding tasks.
- It supports many programming languages.
- It helps developers focus more on problem-solving.

## Impact on Software Development
- GitHub Copilot can improve developer productivity.
- It can help beginners understand coding examples.
- It is becoming part of modern software development workflows.
- AI coding assistants are changing how developers write and review code.
