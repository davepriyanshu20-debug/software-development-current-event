\## Topic: What is GitHub Copilot?



\### What it is

\- GitHub Copilot is an AI-powered coding assistant built by GitHub in partnership with OpenAI.

\- It uses machine learning models trained on large amounts of public code to suggest code completions in real time.

\- Source: https://www.netcomlearning.com/blog/what-is-github-copilot-a-complete-guide



\### How it works

\- Copilot reads the code around your cursor (before and after), plus other open files, to build context.

\- That context is sent to Copilot's underlying model, which makes a probabilistic prediction of what code is likely to come next.

\- Important nuance: Copilot does not "copy-paste" from training data — it generates new suggestions based on patterns it learned.

\- Source: https://github.com/features/copilot



\### How it's evolved ( A good hook intro for us )

\- Originally just inline autocomplete in the editor.

\- Now includes:

&#x20; - \*\*Copilot Chat\*\* — contextual Q\&A inside the IDE

&#x20; - \*\*Cloud agent mode\*\* (GA since Sept 2025) — assign it a GitHub issue, it works in a sandboxed environment, pushes commits to a draft PR, and asks for review

&#x20; - \*\*Copilot CLI\*\* (GA Feb 2026) — command-line tool with autonomous "Autopilot" mode

&#x20; - \*\*Code review\*\* — can review pull requests automatically alongside tools like ESLint/CodeQL

\- Source: https://dev.to/carlosjcastrog/github-copilot-in-2026-is-not-what-you-think-it-is-anymore-ij3



\### IDE support / availability

\- Works in VS Code, Visual Studio, JetBrains IDEs, Neovim, GitHub Codespaces.

\- Pricing (as of June 2026): Individual $10/month, Business $19/user/month.

\- Source: https://contentwave.net/article/github-copilot-2026-review-practical-verdict-for-dev-teams-now



\### Why this matters for the intro

Copilot is a good "current event" topic because it's actively changing in 2026 — moving from simple autocomplete to a full agentic system that can write, review, and even fix code with less human involvement. This sets up the rest of the video (Member 2: benefits, Member 3: risks/limitations).
